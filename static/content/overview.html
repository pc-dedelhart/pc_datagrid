Overview of Nuby-Express

Nuby-Express is built for applications designed to scale.

Action Centric

Focus on actions for rapid full stack production

  • Action Script
  • Client side JS
  • Static Files (CSS, images)
  • Configuration, routing

Built on proven stack

  • the Express.js web framework
  • EJS
  • tap
  • Underscore
  • Mongoose
  • Gate

Designed for Testing

  • Actions can be tested without loading the entire site
  • Actions emit errors through native Node.js eventListeners

Streamlined for rapid development

  • Factories for Actions, Controllers, Components
  • Quick Access to Action-centric files, config
